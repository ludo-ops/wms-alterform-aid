# WMS Simulator – Formation Cariste

## Lancement
1. Enregistre le code dans `index.html`
2. Ouvre le fichier dans Chrome / Edge / Firefox
3. Pour le scan caméra sur téléphone, héberge-le en HTTPS

## Fonctionnalités
- Gestion des stocks
- Réception
- Commandes
- Scanner QR caméra
- Pistolet scanner USB/BT
- QR codes
- PDF commande
- PDF étiquettes
- Mode démo

## Données
Les données sont gardées dans `localStorage`, donc elles restent après fermeture du navigateur, même une semaine plus tard sur le même appareil.
